enum factory::Basic_kernel::Flag
strong

Various kernel flags.

Enumerator
carries_parent 

Setting the flag tells factory::Kernel_stream that the kernel carries a backup copy of its parent to the subordinate node. This is the main mechanism of providing fault tolerance for principal kernels, it works only when principal kernel have only one subordinate at a time.

Deprecated:
This flag should not be used for purposes other than debugging.
